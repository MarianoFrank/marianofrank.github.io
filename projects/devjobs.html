DevJobs

No cuento con hosting para este proyecto pero pasaré a explicártelo.

Este proyecto lo hice para practicar, con la idea de una red social para publicar trabajos como LinkedIn.

Cuenta con el flujo de creación de usuarios típica, con envío de token al email. Una vez creada la cuenta podemos subir las vacantes.

Asi es como se ve la vacante una vez publicada, cuenta con subida de CV en pdf.

A continuación se ve el dashboard de un usuario que publicó algunas vacantes.

Lo usuarios pueden modificar los datos de su cuenta.

Por ultimo asi se ve el inicio donde cualquier persona podrá buscar y postularse a las vacantes.
